Telengard C64 maps

This project is an attempt to reverse-engineer the algorithm used to procedurally generate the fifty levels of Telengard (Commodore 64 version.)

Sample output:

  • Complete level 1
  • Level 1 - around starting location
  • Level 1 - northwest corner
  • Complete level 2
  • Level 2 - under start location
  • Level 2 - northwest corner
  • Partial level 50 (down the elevator from start location.)

Based on my notes, it looks like there are some inaccuracies; however, I do recall successfully using the level 50 map to duck down to level 50, open a puzzle box, and return to the inn with an enormous amount of loot.
